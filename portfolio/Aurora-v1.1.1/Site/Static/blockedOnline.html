Blocked Online

Unveiling internet censorship

I worked on this project with a friend who created an application that allows aware citizens to report on online censorship from different parts of the world. Coming from a country with heavy censorship of the web, I was truly excited to make this project happen.

BlockedOnline uses crowdsourcing approach to gather data on internet censorship. This project relies on people from various countries to download the application from the website, run it on their local machine and thereby provide information on blocked websites in their region. The goal behind BlockedOnline is to provide accessible and readable information on internet censorship in different countries and to raise global awareness of what often remains hidden from the eyes of the public.

Website overview

The landing page presents the user with an interactive map showing statistics on blocked websites around the world, as well as two main action buttons inviting the user to either explore data or get involved in the project.

Under the “Explore data” tab, the user can research censorship facts by URL or by country.

The follwing templates show search results by country and by a website. The structure of the pages allows users to view information quickly, but dive into specific topics if they’re interested.

Under the “Get’ Involved” tab, the user can download the application, learn more about the project and find contact information.
